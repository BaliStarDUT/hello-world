# Lol_Heros
```
Range type：范围类型（如攻击距离、防御距离等）
Resource type：资源类型（如魔法抗性、能量来源等）
Basic HP：基础血量
HP regeneration：血量再生能力
HP regeneration per level：每级增加的血量再生量
Basic mana：基础 mana 值
Mana regeneration per level：每级增加的 mana 再生量
Movement speed：移动速度（如步速、飞行速度等）
Basic armor：基础防御值
Arcade armor per level：游戏模式下每级增加的防御值
Basic magic resistance：基础抗魔法值
Magic resistance per level：每级增加的抗魔法值
Attack range：攻击范围（如单体攻击距离、群体攻击距离等）
HP regeneration per level：每级增加的血量再生量
Basic attack damage：基础攻击力
Attack damage per level：每级增加的攻击力
Attack speed per level：每级增加的攻击速度
Attack speed：攻击速度（如普通攻击、特殊攻击等）
AS ratio：攻击速度比
```
